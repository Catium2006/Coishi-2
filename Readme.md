## 用法
1.克隆仓库  
2.导入IDEA gradle项目  
3.Build -> Build Artifics -> Coishi_with_utf-8.jar -> Build  
4.运行仓库根目录下的run.bat,自动复制出Jar文件并运行-----(~~不会真的有Java开发者不装Java运行环境吧~~)
